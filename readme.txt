Installation
============

1. Upload the cc.* files to their equivalent directories in your OpenCart installation

2. Install the extension in the OpenCart admin panel Extensions -> Payment -> CC Avenue -> Install
3. Ensure module is enabled. To avoid common problems:
   - Please enter your Merchant-ID, not your username or e-mail adress!
   - Optional: Enter a Working_Key for more secure transactions.
	* You must Activat Working_Key sttiengs from CCAvenue panel to use redirect url.
4. Veel succes!
