1. Unpack the sitemap category into the root folder of your opencart and set read and write permissions on it.

2. Put this into your robots.txt: 
Sitemap: http://www.YOURSITE.COM/sitemaps/sitemapindex.xml

3. Run the file http://www.YOURSITE.COM/sitemaps/sitemap.php

4. When done, submit the http://www.YOURSITE.COM/sitemaps/sitemapindex.xml to google
